This GitHub is dedicated to mobile testing. 📱  
Here I publish open tasks for Mobile QA. Tasks include feature documentation, mobile UI mockups, and even full mobile applications with task lists (open projects available to GitHub users).

---

### 📱 Mobile App for Mobile QA "Mobile Testers"

A mobile application designed for mobile QA specialists.  
Includes: tests, case-based tasks, podcasts, interview preparation, and original articles.  
The project is useful for those who want to grow in mobile testing.

A full-featured mobile application for practicing mobile testing.
